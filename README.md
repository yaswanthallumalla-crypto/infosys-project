# infosys-project
This project focuses on analyzing and visualizing food consumption trends using Facebook data to understand user preferences, engagement patterns, and emerging food trends. The dataset includes interactions such as likes, shares, comments, and post reach related to different food categories across regions and time periods.

Using Python, SQL, MS Excel, and Power BI, the data is cleaned, processed, and transformed into interactive dashboards. The visualization highlights popular food items, seasonal trends, regional preferences, and engagement metrics to help businesses, restaurants, and marketers make data-driven decisions.

The project emphasizes analytical storytelling, enabling users to quickly identify insights such as trending cuisines, peak engagement times, and customer interest patterns. The dashboard improves insight generation and reporting efficiency by presenting complex social media data in a simple and visually appealing manner.
